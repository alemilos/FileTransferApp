# Installation (Server)

First run `make`, this will start the server on default address and port.

You can also run `make server`

# Installation (Client)

`make client`

# Usage

./server -a [address] -p [port] -d [root_dir]
./client -a [address] -p [port] [-w | -r | -l] -f [input_file] -o [output_file]
